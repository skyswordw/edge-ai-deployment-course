# 端侧模型量化部署技术专题

本课程面向需要把 AI 模型落到手机、PC、车载、IoT、工业终端、摄像头、机器人等端侧设备的技术团队。课程从端侧部署的问题框架开始，依次展开量化基础、大模型量化、量化精度修复、模型压缩与蒸馏、推理框架选型、VLM 与 Agent 端侧形态，最后用案例串联完整工程路径。

课程的主线不是单独讲某个算法，而是围绕一个工程目标展开：让模型在真实目标设备上稳定达到业务可用标准。

## 学完后应能做到

- 判断端侧部署项目需要优先处理模型能力、设备资源、runtime、功耗散热、网络环境还是产品体验问题。
- 解释 PTQ、QAT、GPTQ、AWQ、SmoothQuant、LLM.int8()、KV Cache 量化等方法的基本思想与适用条件。
- 系统排查量化后精度下降，区分算法问题、数据问题、导出问题、runtime 差异和评估指标问题。
- 把量化与剪枝、低秩分解、参数共享、知识蒸馏、小模型训练和架构重设计放到同一个压缩体系中比较。
- 基于目标设备、runtime、算子覆盖、低比特 kernel、fallback 行为和 profiling 结果选择部署链路。
- 理解 VLM 与 Agent 在端侧的系统组成、瓶颈和端云协同形态。
