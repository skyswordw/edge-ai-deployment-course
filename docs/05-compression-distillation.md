# 专题五 压缩与蒸馏

**时间：14:25-15:10**

## 授课目标

帮助学员把视角从量化扩展到完整的模型压缩体系，理解剪枝、低秩分解、参数共享、知识蒸馏、小模型训练和架构重设计如何共同服务于端侧部署。学员应能够判断什么时候适合压缩现有模型，什么时候应该选择或训练更端侧友好的模型。

## 核心内容

本专题通过讲解模型压缩的主要技术路线，介绍量化、剪枝、低秩分解、参数共享、知识蒸馏和架构重设计之间的关系。课程将说明非结构化剪枝虽然理论压缩率高，但未必带来真实端侧加速，而结构化剪枝更容易适配端侧硬件，但需要配合 fine-tuning 和真实设备 profiling 才能判断收益。

本专题还将介绍知识蒸馏在端侧部署中的作用。蒸馏不仅可以把大模型能力迁移到小模型，也可以用于修复量化后的能力下降。传统视觉模型可以蒸馏 logits、feature 或 attention，LLM 可以蒸馏 response 或 preference，VLM 则需要分别关注 vision encoder、projector 和 LLM 的能力迁移。课程也会讨论为什么有时直接选择更合适的模型架构，比压缩一个不适合端侧的模型更有效。
