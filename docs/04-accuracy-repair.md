# 专题四 量化精度修复

**时间：13:30-14:25**

## 授课目标

帮助学员掌握量化后精度下降的系统排查方法，能够从 baseline、数据预处理、模型导出、runtime 实现、校准数据、敏感层、outlier 和评估指标等角度定位问题。学员应建立先确认问题、再定位原因、最后选择修复手段的工程思维。

## 核心内容

本专题通过讲解量化后精度下降的常见表现，介绍分类检测指标下降、生成质量变差、格式错误增多、VLM 细粒度识别能力退化和 Agent 规划稳定性下降等问题形态。课程将强调，精度下降不一定来自量化算法本身，也可能来自 baseline 不可靠、预处理后处理不一致、模型导出错误、runtime 行为差异或评估指标不一致。

本专题还将介绍量化精度修复的工程流程，包括校准集重构、逐层敏感性分析、逐层回退、mixed precision、敏感层保留高精度、clipping 和 percentile 调整、activation outlier 处理，以及必要时使用 QAT、LoRA、Adapter 或蒸馏进行补偿。所有修复都需要回到目标设备重新 profiling，确认精度、延迟和内存收益是否真实成立。
