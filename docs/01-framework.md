# 专题一 端侧部署问题框架

**时间：09:00-09:30**

## 授课目标

帮助学员建立端侧模型部署的整体判断框架，理解端侧 AI 落地需要同时处理模型能力、设备资源、运行时框架、功耗散热、网络环境和产品体验之间的平衡。学员应能够明确量化、压缩、蒸馏和推理框架选型分别解决什么问题，并把后续技术内容放到统一的工程目标中理解。

## 核心内容

本专题通过讲解端侧推理重新受到重视的背景，介绍云端推理在隐私、延迟、弱网、成本和个性化体验方面的限制，以及手机、PC、车载、IoT、工业终端、摄像头和机器人等典型端侧场景。课程将引导学员理解端侧部署并不是把云端模型直接搬到设备上运行，而是要在模型能力与设备资源之间建立可落地的平衡关系。

本专题还将介绍端侧部署的关键评价维度，包括任务精度、端到端延迟、首 token 延迟、内存峰值、KV Cache 占用、功耗、发热降频、稳定性和维护成本。通过这些内容，学员可以理解后续量化、压缩、蒸馏和推理框架选型都服务于同一个目标，也就是让模型在真实目标设备上稳定达到业务可用标准。
